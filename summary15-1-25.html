this is a summary of the class on 15/1/25

h1, h2, h3 are used as titles!

ther are 6 sizes of headers (titles), as the number following h goes up, the size goes down

Example:

h1

h2

h3

h4

h5
h6

in order to use a header, you need to write in your software this code:

<h1> (your title) </h1>

theres also list tags!

to use lists you will first have to define whether its unordered or ordered list

<ul> (list title) </ul> = unordered lists

lists that are unordered have their items shown with a * before them

<ol> (list title) </ol> = ordered lists

lists that are ordered are numbered by 1,2,3 and so on (1.item1 2.item2)

theres probably more types, but we only studied those two so far

customization

theres an option to customize headers, paragraps, even the background!

order to do that, before you close the header with >, you need to add style=.

to change the color, its style="color= (color name/rgb/# tag);"

example of how to code a red title:

<h1 style="color: red;"> Title </h1> Title

color isnt the only thing you can customize, you can also customize where the text is

for example, if you want a title to be in the center, the code would look like this:

<h1 style="text-align: center;"> Title

to change background colors its as simple as going into <body> and adding style there

if you'd like to combine the two, it would look something like this:

<h1 style="color: red; text-align: center;"> Title

when coded, the execution will look like this:

Title

for text you can also choose the font size, font family, size, and so on

to change the font its style="font-size: 24px;"

the px (pixels) unit is used in this demonstration, but there are more units:

px, em, rem, %,, vw, and more...

another different way to change would be <style>,

its done under <title> inside <head>.

it affects the deafult setting of types, unless you specify it

thats how you should change it for convenience, in order to do it you do this:

<style>

h1{

color: red;

text-align: center;

font-size: 24px;

}

</style>

dont forget that you will have to specify changes, and stuff shown here would affect all h1's

unless coded otherwise.
